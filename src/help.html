Getting Started

Welcome to the Audio Guide! This guide will help you navigate through the artwork collection and make the most of the available features.

Ways to Access Artworks

  • Number Input

    Simply tap the input field and use the keypad to enter the artwork number.

  • QR Code Scanner

    Use the QR scanner button to scan artwork QR codes for quick access.

  • Gallery View

    Browse all artworks in the gallery view and click on any artwork to view details.

Audio Player Features

  • Playback Controls

    Play, pause, and navigate through the audio content using the player controls.

  • Language Selection

    Switch between available languages at any time using the language selector.

  • Image Gallery

    View multiple images of each artwork using the image slider. Use arrows or swipe to navigate.

Accessibility

  • Dark Mode

    Toggle between light and dark themes for comfortable viewing in any environment.

  • Clear Text

    Large, readable text and high contrast colors for better visibility.
